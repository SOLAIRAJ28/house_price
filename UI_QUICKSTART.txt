üé® CUSTOMIZATION TIPS:

Change Gradio Theme (in main.py):
  interface = gr.Blocks(theme=gr.themes.Default())
  
  Other options:
  - gr.themes.Soft() (current)
  - gr.themes.Default()
  - gr.themes.Monochrome()
  - gr.themes.Glass()

Change HTML Colors (in index.html):
  Look for "linear-gradient" in CSS section
  Change: #667eea (blue) and #764ba2 (purple)
  
Adjust Input Ranges:
  Edit slider min/max values
  In HTML: min="32.5" max="42"
  In Gradio: gr.Slider(32.5, 42, ...)

üîß TROUBLESHOOTING:

Issue: "Address already in use"
Fix: Change ports in main.py
  uvicorn.run(app, host="0.0.0.0", port=10001)
  interface.launch(server_port=7861)

Issue: "Module not found: gradio"
Fix: Install dependencies
  pip install -r requirements.txt

Issue: "Model file not found"
Fix: Ensure house_model.pkl exists
  python train_model.py

Issue: "Cannot connect to API"
Fix: Check FastAPI is running on port 10000
  Check console for error messages
